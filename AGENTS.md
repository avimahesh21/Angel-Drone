## Project Summary
A clean, minimalistic drone escort application UI built with Next.js, mimicking the Uber aesthetic. The app allows users to request a drone to escort them home for safety, featuring a simulated map and real-time state transitions from searching to escorting.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS 4
- Icons: Lucide React
- Animations: Framer Motion
- UI Components: Radix UI, Vaul (Drawer)

## Architecture
- `src/app`: Page routes and layouts.
- `src/components`: Reusable UI components.
- `src/lib`: Utility functions and shared logic.
- `src/hooks`: Custom React hooks for state management.

## User Preferences
- Clean and minimalistic design.
- Mimic Uber's UI/UX.
- Use Next.js.

## Project Guidelines
- Prefer named exports.
- Use functional components.
- Maintain a cohesive aesthetic with sharp accents.
- No comments unless requested.

## Common Patterns
- Simulated map backgrounds for demo purposes.
- Bottom sheet (drawer) for main user interaction flow.
